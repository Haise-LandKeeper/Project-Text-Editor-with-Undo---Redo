# Text Editor Undo/Redo (C++)

Chương trình mô phỏng đơn giản chức năng Undo/Redo trong trình soạn thảo văn bản.

Sử dụng các cấu trúc dữ liệu cơ bản:
- Stack
- Queue
- Double Linked List

## Chức năng
- Nhập dữ liệu
- Undo / Redo
- Hiển thị nội dung hiện tại
- Xem lịch sử thao tác

## Mô tả
Chương trình được xây dựng nhằm mục đích học tập, giúp minh họa cách áp dụng các cấu trúc dữ liệu vào bài toán thực tế như Undo/Redo.

## Run
Compile file `.cpp` và chạy trên môi trường console.
